最后一名主犯被抓！21年前黑龙江灭门命案尘埃落定

阅读：35k • 新晚报 • 2019-05-25 00:33:51

1998年10月的一天晚上，

黑龙江的黄某像往常一样，

一家三口吃过晚饭后看电视。

这时传来了敲门声，

他没有想到，

这次开门给他带来了杀身之祸。

网络配图

日前，烟台开发区公安分局刑侦大队在烟台成功抓获一名逃犯，随着这最后一名主犯的落网，历时21年之久、震惊一时的黑龙江灭门惨案终于尘埃落地。

//

夜晚一家三口倒在血泊中

//

听到敲门声后，黄某起身去开门。刚打开房门，就冲进来两名男子。

  • 一名男子持刀捅向黄某；

  • 另一名男子进门后冲向了黄某的老婆和孩子，两人相继被刺倒地。

一眨眼的功夫，三个鲜活的生命都倒在了地上。

两名男子行凶后，迅速逃离了现场。邻居发现黄某一家三口被捅后，遂即报警，黄某一家三口终因失血过多，经医院抢救无效死亡。

此案在黑龙江震惊一时。

//

嫌疑人作案后“人间蒸发”

//

案件发生后，当地引起了极大恐慌，公安机关立即展开侦查。

经调查了解，死者黄某系社会无业人员，为人贪财、好色、嗜赌，有许多不良嗜好，社会关系也很复杂。黄某利用承揽某综合市场管理工作之便，招集社会闲散人员，欺压市场商贩，收取管理费用，插手各种矛盾纠纷，从中渔利，与多人都有结怨。

网络配图

警方做了大量工作，最终确定了两名犯罪嫌疑人，但嫌疑人作案发后早已人间蒸发，不知去向。警方经过缜密侦查后发现，于某与犯罪嫌疑人之一的张某认识，而且他与死者黄某也有矛盾，所以警方对于某开展工作。

经讯问，于某承认他曾经跟张某说想教训一下黄某，但后来发生了什么，他不知道。由于主要犯罪嫌疑人张某尚未到案，无法进行佐证，当地法院以故意伤害罪判处于某有期徒刑六年。

//

天不藏奸 真相浮出水面

//

2019年1月，经过多方努力，犯罪嫌疑人张某被广东警方成功抓获，随着张某落网，21年前尘封的灭门案真相终于大白于天下——

1998年10月的一天，于某找到张某，说自己摆摊卖调料时经常被黄某欺负，想雇佣张某杀死黄某一家三口。张某由于急需用钱，没有迟疑，毫不犹豫地答应下来。

网络配图

张某收了于某的钱后，约了同乡王某残忍的将黄某一家三口全部杀害，包括黄某年仅9岁的孩子。张某和王某合伙作案后，便约定老死不相往来，从此分道扬镳，开始了逃亡生活。

根据张某提供的线索，警方很快在四川将同案犯罪嫌疑人王某抓获。于某指使张、王两人，制造灭门大案终于真相大白，但此时于某已经出狱不知所踪，黑龙江警方将于某列为网上逃犯。

黑龙江警方发现，于某2004年出狱后，携妻女前往烟台定居，遂向烟台开发区公安分局发出协查通报。最终，烟台开发区公安分局刑侦大队民警成功将于某抓获。

抓捕时，于某没有做任何反抗，只是询问民警，张某被你们抓了吧？得到印证后，他放声痛哭。

至此，该案三名嫌疑人均已到案。于某已被开发区警方依法移交给黑龙江警方。

天网恢恢，疏而不漏。
